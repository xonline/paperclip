# Social Media

Create platform-native content that sounds like a real human, not an AI assistant. Each platform has its own voice, format, and audience expectations — match them.

## When to Use

- writing captions, posts, threads, or bios for any platform
- building a content calendar or batch of posts
- repurposing a blog post, video, or idea across multiple platforms
- hashtag research and strategy
- writing hooks or opening lines
- reviewing/improving existing social copy

## Platform Cheat Sheet

### Instagram
- **Caption length**: short (1-3 lines) for lifestyle/product, longer (3-8 lines) for storytelling or education
- **Hook**: first line must stop the scroll — question, bold claim, or surprising statement
- **Format**: line breaks between paragraphs, no walls of text
- **Hashtags**: 3-10 targeted; mix niche + broad; put at end or first comment
- **CTA**: always end with one — "link in bio", "save this", "tag someone", "comment below"
- **Emojis**: sparingly for punctuation or visual breaks, not decorative filler

### Facebook
- **Caption length**: medium to long — Facebook users read more
- **Format**: conversational, community-first
- **Engagement bait (good kind)**: ask questions, create polls, invite opinions
- **Links**: paste link in post body (not just comments) — Facebook shows preview card
- **Tone**: warmer and more personal than LinkedIn

### Twitter / X
- **Length**: under 280 chars for single tweets; threads for longer ideas
- **Thread format**: hook tweet first, numbered or flowing, end thread with CTA or summary
- **Tone**: direct, witty, or punchy — no corporate fluff
- **No hashtags spam**: 1-2 max per tweet
- **Reply hooks**: write tweets that beg for replies or quote-tweets

### LinkedIn
- **Format**: short first line (hook), then expand after "...more"
- **Structure**: 3-5 punchy paragraphs with line breaks, no long blocks
- **Tone**: professional but personal — first-person stories, lessons learned, opinions
- **No jargon**: write for smart generalists, not insiders
- **CTA**: end with a question to drive comments
- **Hashtags**: 3-5, relevant and specific

### TikTok / Reels caption
- **Very short**: 1-2 lines max — the video does the work
- **Hook in the caption** = the text overlay or spoken line the video opens with
- **Trending sounds/hashtags**: note where they apply
- **Tone**: casual, fast, native to Gen Z/Millennial culture without trying too hard

## Content Frameworks

### Hook formulas
```
- "Most people don't know that [surprising fact]..."
- "I used to [old way]. Now I [new way]. Here's why:"
- "Hot take: [contrarian opinion]"
- "[Number] things nobody tells you about [topic]:"
- "Stop doing [common mistake]. Do this instead:"
- "The [outcome] most people spend years chasing can be done in [short time] — here's how:"
```

### Caption structure (Instagram/Facebook)
```
[Hook — 1 line]

[Body — the value, story, or insight]

[CTA — 1 line]

[Hashtags]
```

### LinkedIn post structure
```
[Bold claim or personal observation — 1 line]

[Expand with story or context — 2-3 short paragraphs]

[Lesson or takeaway]

[Question to audience]

#relevant #hashtags
```

### Thread structure (Twitter/X)
```
Tweet 1: Hook — the promise of the thread
Tweet 2-N: One idea per tweet, clear and direct
Last tweet: Summary + CTA (follow, retweet, link)
```

## Content Types

| Type | Best for | Key rule |
|---|---|---|
| Educational | Build authority | Give real value, not fluff |
| Behind-the-scenes | Build trust | Authentic > polished |
| Social proof | Convert | Specific results beat vague claims |
| Storytelling | Engagement | Conflict → resolution arc |
| Promotional | Drive action | 1 promo per 4-5 value posts |
| Trending/reactive | Reach | Only join if genuinely relevant |
| UGC repost | Community | Always credit the original |

## Content Calendar

When asked to create a content calendar:
1. Ask: platform(s), posting frequency, content pillars (or infer from brand/niche)
2. Use 80/20 rule: 80% value/entertainment, 20% promotional
3. Mix content types — don't repeat same format back-to-back
4. Output as a table: Date | Platform | Type | Hook/Topic | Notes

```
| Date | Platform | Type | Hook / Topic | Format |
|------|----------|------|--------------|--------|
| Mon  | Instagram | Educational | 3 things about X | Carousel |
| Tue  | LinkedIn | Story | How we solved Y | Post |
| Wed  | Twitter  | Opinion | Hot take on Z | Thread |
```

## Hashtag Strategy

- **Niche tags** (< 500K posts): higher chance of ranking, targeted audience
- **Mid tags** (500K–5M posts): good reach/competition balance
- **Broad tags** (> 5M posts): exposure but competitive
- Mix all three — don't use only mega-tags
- Research: think about what your ideal follower actually searches
- Avoid banned or overused spam tags

## Writing Rules

- Write in the brand's voice — check `brand-voice` skill if a voice profile exists
- No AI-sounding phrases: "dive into", "delve", "let's explore", "in the realm of", "game-changer" (unless ironic)
- Contractions make copy sound human — use them
- Short sentences > long ones on social
- One idea per post — don't try to say everything
- Read the copy aloud — if it sounds weird, rewrite it

## Image & Creative Briefs

When asked to create accompanying visuals:
- Use `image-prompt` skill to generate Nano Banana prompts for the visual
- Describe: subject, mood, colour palette, style, text overlay (if any)
- Match the platform aspect ratio: Instagram feed 1:1 or 4:5, Stories/Reels 9:16, LinkedIn 1.91:1, Twitter 16:9
